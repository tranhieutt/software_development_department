## 4. Cross-Cutting Themes

1. **"Spec ahead of implementation" pattern.** CLAUDE.md, coordination-rules.md, and the Rule 14 circuit-breaker assume artifacts (ADRs, `active.md`, handoffs) that never materialized. The governance layer writes checks the runtime cannot cash.
2. **Legibility debt, not capability debt.** The system already *has* memory, orchestration, and safeguards; they are scattered across 5+ files. ¬ß16 of the spec correctly identifies this ‚Äî the audit confirms it. Consolidation artifacts (¬ß15) > new features.
3. **Implicit coupling in the hook layer.** Multiple hooks touch the same file (ledger, memory, session logs) without a documented ordering or owner.
4. **Stale legacy fragments.** Decision ledger mixes schema versions; PS1 hooks ghost the bash ones; `report_upgrade_ver4_opus47.md` sits in the repo root.
